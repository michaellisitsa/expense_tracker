How to use?

This website allows you to create categories of expenses, and then submit your expenses over some periods of time (which can be with gaps).
Then the website will work out how much you have spent (averaging over any missing periods) in the last week / month / year.

  1. Add some new expenditure categories, and an approximate budget you have for each.
  2. Click into the category you are interested in.
  3. Create a new time expense period. This is sort of like an order, which will have one or more individual items inside it.
  4. You'll be directed to a list where you can add as many individual expense items under the time period.
  5. The top banner will always show an average of you expenditure for the last week / month / year, as well as what you budgetted.
